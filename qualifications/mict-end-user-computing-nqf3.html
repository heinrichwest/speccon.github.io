Qualification Overview

The National Certificate: Information Technology: End User Computing is designed to build the knowledge and skills required for confident operation in End User Computing environments. This qualification empowers learners to acquire essential ICT skills for the modern workplace and provides recognition for workplace experience.

Exit Level Outcomes

Upon completion of this qualification, learners will be able to:

  • Apply Word Processing skills for document creation and editing
  • Apply Presentation Application skills for effective presentations
  • Apply Spreadsheet Application skills for data analysis and calculations
  • Apply Electronic Mail skills for professional communication
  • Apply Web Browser skills for internet research and navigation
  • Understand ICT's impact on organizations and society
  • Improve communication and mathematical literacy in ICT contexts

Target Roles

This qualification is ideal for:

Personal Assistants
Office Administrators
Project Administrators
Data Capturers
Reception Staff
General Office Workers

Core Application Areas

Word Processing Applications

  • • Create, edit, and format professional documents
  • • Work with tables, headers, and footers
  • • Use templates and styles for consistency
  • • Mail merge and document collaboration
  • • Print and share documents electronically

Spreadsheet Applications

  • • Create and format spreadsheets and workbooks
  • • Use formulas and functions for calculations
  • • Create charts and graphs for data visualization
  • • Sort, filter, and analyze data
  • • Print and share spreadsheets effectively

Presentation Applications

  • • Design and create professional presentations
  • • Work with slides, layouts, and themes
  • • Insert multimedia elements and animations
  • • Deliver effective presentations
  • • Export and share presentations

Electronic Mail Applications

  • • Compose, send, and receive professional emails
  • • Manage contacts and address books
  • • Organize emails with folders and rules
  • • Use calendar and scheduling features
  • • Email etiquette and security best practices

Web Browser Applications

  • • Navigate websites and use search engines effectively
  • • Bookmark and organize favorite websites
  • • Download and upload files safely
  • • Understand internet security and privacy
  • • Use web-based applications and services

ICT Fundamentals

  • • Understand basic ICT concepts and terminology
  • • File management and organization
  • • Computer hardware and software basics
  • • Digital citizenship and ethics
  • • Troubleshooting common computer problems
