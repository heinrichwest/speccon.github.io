Qualification Overview

The Further Education and Training Certificate: Information Technology: System Support qualification provides essential IT skills to install, maintain, and troubleshoot hardware, software, and networks for efficient, secure operations. This comprehensive program combines theoretical knowledge with hands-on practical experience in client-server networking, database management, LAN design, and multi-user operating systems.

Learning Outcomes

Upon completion of this qualification, learners will be able to:

  • Install and configure multi-user networked operating systems
  • Administer and maintain local area computer networks (LANs)
  • Design LANs for departmental office environments
  • Create database access using structured query language (SQL)
  • Administer security systems for multi-user computers
  • Demonstrate appropriate customer care according to Service Level Agreements

Core Modules

Module 1: Personal Development

Research computer topics, work in project teams, resolve OS problems, conduct technical meetings, and understand business principles and IT ethics.

Module 2: Client-Server Networking

Understand LAN management issues and multi-user computer operating systems concepts.

Module 3: Server Administration

Configure and operate server peripherals, administer LANs, install/configure/monitor networked operating systems, and test IT systems.

Module 4: Database Access

Create database access using SQL, understand database management systems, and administer security for multi-user computers.

Module 5: LAN Design & Development

Demonstrate customer care per SLA, describe enterprise systems management, and design LANs for departmental office environments.

Module 6: Network Architecture & Standards

Understand network architectures/standards, install networked workstations, compare WANs with LANs, and commission LANs.
